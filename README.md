
## 项目介绍

Simple-Es是一个快速处理聚合查询ES的工具包，可以向使用Mybatis-plus一样方便快捷的处理聚合查询相关问题。

## 项目结构

```
simple-Es -- 父工程
├── simple-es-annotation -- 注解模块
├── simple-es-boot-starter -- 整合Spring Boot
├── simple-es-common -- 基础模块
├── simple-es-core -- 核心
├── simple-es-extension -- 扩展
├── simple-es-parent -- 包管理
├── simple-es-sample -- 案例
└── simple-es-test -- 测试
```
